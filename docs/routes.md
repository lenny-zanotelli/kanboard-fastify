# Project routes

### List

| method | ressource    | action                   |
| ------ | ------------ | ------------------------ |
| GET    | `/lists`     | Return all lists         |
| GET    | `/lists/:id` | Return a list with an id |
| POST   | `/lists`     | Create a list            |
| PUT    | `/lists/:id` | Update a list with an id |
| DELETE | `/lists/:id` | Delete a list with an id |
